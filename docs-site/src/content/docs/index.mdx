`ausecon-mcp-server` exposes read-only MCP tools for official Australian economic data. It is
designed for analysts, economists, and MCP clients that need reliable discovery, provenance-rich
retrieval, and stable JSON outputs without turning the server into a general economic chatbot.

## What it provides

- Dataset discovery across curated Australian Bureau of Statistics (ABS), Reserve Bank of
  Australia (RBA), and Australian Prudential Regulation Authority (APRA) entries.
- Analyst-friendly semantic concepts such as CPI, GDP growth, unemployment, cash rate, yields,
  credit, APRA banking and insurance ratios, and external-sector indicators.
- Source-native retrieval tools for exact ABS SDMX keys, RBA statistical tables, and curated APRA
  publications.
- Normalised retrieval responses with explicit metadata, series descriptors, observations, and
  server-version provenance.
- Read-only MCP resources and prompt templates for common economist workflows.

## Current status

Version `1.10.0` is the current release line. The server remains intentionally selective: it covers
the main analyst workflows, supports stdio plus hosted Streamable HTTP, and does not aim to mirror
every ABS, RBA, or APRA release.

## Quick install

```bash
uvx ausecon-mcp-server
```

The server speaks MCP over standard input/output and is normally launched by your MCP client.
