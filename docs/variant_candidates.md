# Variant candidates

This document is regenerated by `scripts/dump_variant_candidates.py`. It lists
the series rows exposed by every curated RBA table and the dimensions and
codelists exposed by every curated ABS dataflow, so the maintainer can
hand-populate `variants[*].rba_series_ids` and `variants[*].abs_key` ahead of
the v0.4.0 semantic resolver.

Regenerate with:

    uv run python scripts/dump_variant_candidates.py

The script hits the live ABS and RBA endpoints; expect one HTTP request per
catalogue entry.
